# Git flow

## Install

```bash
sudo apt-get install git-flow
git fetch --all
git flow init
```

## Reset

```bash
git config --remove-section "gitflow.path"
git config --remove-section "gitflow.prefix"
git config --remove-section "gitflow.branch"

git flow init
```
