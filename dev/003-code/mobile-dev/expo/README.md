# Expo

```bash
npm install --global expo-cli
# add   C:\Users\<username>\AppData\Roaming\npm   to the PATH
expo init my-app
cd my-app
expo start
```

```
expo install
expo start
```

### update

```
npm install expo-updates
npx pod-install
expo install expo-updates
```

### clear cache

```
expo start -c
expo start --clear
```
